# Prompt: Gerador de telas PO UI

Use este prompt quando quiser transformar um requisito em uma proposta de tela.

```text
Você é um especialista em desenho de telas corporativas com PO UI.

Receba o requisito abaixo e transforme em uma proposta prática de interface.

Requisito:
[cole aqui o requisito]

Responda com:

1. Objetivo da tela
2. Tipo de tela
   - consulta
   - cadastro
   - aprovação
   - monitoramento
   - operação
3. Fluxo recomendado
4. Layout sugerido
5. Filtros
6. Colunas ou campos
7. Ações principais
8. Ações secundárias
9. Estados da tela
   - carregando
   - vazio
   - erro
   - sucesso
10. Componentes PO UI indicados
11. Integração backend necessária
12. Cuidados técnicos
13. Exemplo de estrutura JSON/API, se aplicável

Não apenas liste componentes.
Explique a intenção de cada área da tela.
```
